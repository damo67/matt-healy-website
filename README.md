# Matt Healy Advisory Website

A clean, modern single-page website for Matt Healy Advisory. Built with static HTML, CSS, and vanilla JavaScript.

## Features

- Single-page scrolling design
- Responsive and mobile-friendly
- Minimalist, professional aesthetic
- Smooth animations and transitions
- Four key service pillars: Advice, Engagement, Reporting, Consultation

## Structure

- `index.html` - Main HTML document
- `styles.css` - All styling and responsive design
- `script.js` - Smooth scroll and scroll animations

## Hosted on GitHub Pages

Visit: [matt-healy-website](https://damo67.github.io/matt-healy-website/)

## Design

- Color Palette: Greys, blacks, whites with blue accents
- Typography: System fonts for performance and consistency
- Responsive: Optimized for desktop, tablet, and mobile
